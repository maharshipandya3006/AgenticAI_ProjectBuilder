# AgenticAI_ProjectBuilder
AgenticAI ProjectBuilder is a multi-agent system designed to automate the creation of software projects from a single user prompt. It leverages a Planner–Architect–Coder workflow to transform high-level ideas into complete, functional codebases.

# 🏗️ Architecture
The system operates using a stateful graph managed by LangGraph, where each node represents a specialized agent:

**Planner Agent:** Receives the initial user prompt and formulates a high-level engineering plan. This includes defining the project name, description, tech stack, features, and a list of necessary files.

**Architect Agent:** Takes the plan and breaks it down into a detailed, ordered sequence of implementation tasks. Each task specifies the file to be modified and provides a granular description of the code to be written, including dependencies and integration points.

**Coder Agent:** A ReAct (Reasoning and Acting) agent that executes the implementation tasks one by one. It uses a set of tools to interact with the file system, such as reading files, writing code, listing the project structure, and running shell commands. It iterates through the tasks until the project is complete.

The entire process is orchestrated to ensure that dependencies are handled correctly and the final output is a coherent and runnable project.


# Example Output
The repository includes an example of a project generated by the agent in the generated_project/ directory. Given the prompt for a simple calculator, the agent created a complete web application with the following files:

index.html: The HTML structure for the calculator.

style.css: CSS for styling the calculator's appearance and layout.

script.js: JavaScript for handling user input, performing calculations, and updating the display.

README.md: A detailed README for the generated calculator project.
